# cidades-api
Construindo uma API Rest de consulta da distancia de cidades do Brasil do zero até a produção. 

Projeto desenvolvido seguindo as aulas do digital innovation one. 

Utilizado nesse projeto:
- Gradle
- Git
- Java 8
- Docker
- PostgreSQL
- Heroku CLI
- Spring Boot
